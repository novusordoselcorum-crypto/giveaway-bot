# 🏎️ Giveaway Bot

Телеграм-бот для розыгрыша с оплатой через Lava Top.

## 🚀 Быстрый старт

### 1. Создай бота
- Напиши @BotFather → `/newbot`
- Сохрани токен

### 2. Настрой config.py
```python
BOT_TOKEN = "твой_токен_от_botfather"
ADMINS = [твой_числовой_id]  # узнай у @userinfobot
OPEN_CHANNEL_URL = "https://t.me/твой_открытый_канал"
CLOSED_CHANNEL_ID = -100...  # ID закрытого канала
LAVA_TOP_URL = "https://pay.lava.top/твоя_ссылка"
```

### 3. Деплой на Railway
1. Залей код на GitHub
2. Зайди на railway.app
3. New Project → Deploy from GitHub
4. Выбери репозиторий
5. Railway сам установит зависимости и запустит

### 4. Настрой Lava Top
1. В личном кабинете Lava Top найди Webhook URL
2. Укажи: `https://твой-проект.railway.app/webhook/lava`
3. Скопируй Secret Key в config.py

## 📱 Команды админа

| Команда | Описание |
|---------|----------|
| `/stats` | Статистика бота |
| `/broadcast текст` | Рассылка всем |
| `/broadcast_paid текст` | Рассылка оплатившим |
| `/set_video` | Ответь на видео чтобы установить |
| `/admins` | Список админов |
| `/add_admin ID` | Добавить админа |
| `/confirm ID` | Ручное подтверждение оплаты |
| `/help` | Справка |

```bash

```

## 📝 Как поменять тексты

### Вариант 1: Через config.py
Открой файл, измени переменные `WELCOME_TEXT`, `CLOSED_CHANNEL_TEXT` и т.д.

### Вариант 2: Через бота (скоро)
`/set_welcome новый текст`

## 🔐 Как добавить бота в закрытый канал

1. Создай закрытый канал
2. Добавь бота как администратора
3. Дай права: "Добавлять участников" + "Приглашать по ссылке"
4. Узнай ID канала: перешли сообщение из канала боту @getidsbot
5. Добавь ID в `CLOSED_CHANNEL_ID` (с минусом! например: -1001234567890)

## ⚠️ Важно

- Бот должен быть админом закрытого канала
- Lava Top вебхук должен быть настроен правильно
- Храни секретные ключи в безопасности!
